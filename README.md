# typing
## タイピングアドベンチャーアプリ

### 始め方
１．Rubyの導入をして,このプロジェクトをダウンロードorクローンしてください。  
  
２．以下のパッケージをgem installしてください。  
`rdbi`  
`rdbi-driver-sqlite3`  
  
３．インストール後,typingディレクトリ配下で以下のコマンドを実行後,ゲームがスタートします。  
`ruby missionx.rb`
